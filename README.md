## ✨ Features

- 🔍 **Search by name** — Look up any Pokémon from all generations
- 📊 **Base stats** — Health, Attack, Defense, Sp. Atk, Sp. Def, Speed
- ⚡ **Fast & lightweight** — No dependencies, just Python

---

## 🔌 How It Works

The app calls the free [PokéAPI](https://pokeapi.co/). No API key is needed.
The returned JSON is parsed to extract stats, which are then rendered in the Tkinter UI.

---

## 🛠️ Usage

1. **Launch** the program with `python main.py`
2. **Type** a Pokémon name (e.g. `charizard/Charizard`) in the search bar
3. **Click** the **Search Pokedex** button
4. **View** the Pokémon's stats!
